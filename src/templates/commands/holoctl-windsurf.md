description: Initialize or inspect holoctl in the current project directory
---

Steps:
1. Run `holoctl doctor` to check if `.holoctl/` is already initialized.
2. If not initialized: run `holoctl init`, then `holoctl compile --target windsurf`.
3. If already initialized: run `holoctl board ls` and show the board status.

Show all command output to the user.
